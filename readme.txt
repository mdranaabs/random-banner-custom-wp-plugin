Random Banner ads WordPress plugin provides users with high level of flexibility to show image banners ads randomly

== Description ==
Random Banner Ads WordPress plugin provides users with high level of flexibility to show image banners, SWF banners and script ads randomly

= Features =

* Supported Banner types: Image

* Add individual link to Image.

* Banners will be displayed randomly based on interval times.

* Banner can be placed almost anywhere using shortcode or widget

* Categories the banners.

* Filter the ads by its category in "Add Banner location"

= How to add Shortcode? =

= In PHP =
`
<?php echo do_shortcode('[random_image]'); ?>
`
= Inside the Posts =
`
[random_image]
`
= Options to use in the shortcode =

`[random_image category="your-category-slag"]`

= Values for options =
* category = default / any category
* delay = 3000 (in Milli Seconds)

== Installation ==

1. Upload the "random-banner-ads" directory to the plugins directory.
2. Go to the plugins setting page and activate "Random Banner Ads"
3. Go to Random Banner Ads and Upload banner by image or URL and add the link for each image
6. Do save.

For more information : [Random Banner Support](https://github.com/mdranaabs)
